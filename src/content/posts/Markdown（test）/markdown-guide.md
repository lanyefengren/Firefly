# Markdown 速查手册

## 标题

```md
# 一级标题
## 二级标题
### 三级标题
```

## 强调

```md
**粗体**  *斜体*  ~~删除线~~  `行内代码`
```

## 列表

```md
- 无序列表项
- 无序列表项

1. 有序列表项
2. 有序列表项
```

## 链接与图片

```md
[链接文字](url)
![图片描述](图片url)
```

## 代码块

````md
```语言名
代码内容
```
````

## 引用与分割线

```md
> 引用内容

---
```

## 表格

```md
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 值A | 值B | 值C |
```

## 任务列表

```md
- [x] 已完成
- [ ] 未完成
```

## 数学公式

```md
行内公式：$E = mc^2$

块级公式：
$$
\sum_{i=1}^n x_i
$$

化学式：$\ce{H2O}$
```

> **核心语法**：标题、列表、代码、链接、图片、表格 — 记住这几个就能覆盖 90% 的写作场景。
